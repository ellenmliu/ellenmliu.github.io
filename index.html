About

  Here's a little about me. I moved to Northern California where I currently teach kids at MV Code how to code. I graduated from UC Irvine with a Bachelor's in Computer Science Engineering. My first language is Java and along the way I picked up other languages like C/C++, JavaScript, and HTML and CSS.

  It's been a journey to find out what I want to do. I went through many routes, but as I was teaching kids all the different platforms, I realized I had a passion for web development. I started some online courses, and began to experiment with what I know. Soon enough, I've made a few web applications and became obsessed with improving on the functionality and aesthetics. I can't wait to learn more to become a front-end web developer.
